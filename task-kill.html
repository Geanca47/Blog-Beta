taskkill — Visual con colores

Copia los comandos a CMD/PowerShell. Aquí ves cómo se ven las respuestas típicas (éxito / error / alternativas).

Comando (éxito):
taskkill /F /IM electron.exe

Salida simulada:

SUCCESS: The process "electron.exe" with PID 4520 has been terminated.
Comando (no encontrado):
taskkill /F /IM electron.exe

Salida simulada:

ERROR: The process "electron.exe" not found.
Alternativas (si no funciona):
taskkill /F /IM Code.exe
taskkill /F /IM code.exe
taskkill /F /IM "Code Helper.exe"

Consejo: Usá /T para terminar procesos hijos:

taskkill /F /IM electron.exe /T
Opción bruta (matando por nombre parcialmente coincidente):
for /f "tokens=2" %i in ('tasklist ^| findstr /i electron.exe') do taskkill /F /PID %i

(En un .bat usa %%i en lugar de %i). Si preferís, abrí el Administrador de tareas → Pestaña Detalles → buscá code.exe o electron.exe → End task.
