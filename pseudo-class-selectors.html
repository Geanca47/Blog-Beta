Pseudo-class Selectors en CSS

:hover - Pasar el mouse sobre un elemento

.btn-hover:hover { background-color: #764ba2; }

:active - Presionar un elemento

.btn-active:active { transform: scale(0.95); }

:focus - Enfoque en un elemento

.input-focus:focus { border-color: #667eea; }

:visited - Enlaces visitados

a:visited { color: #764ba2; }
Google (azul si no lo visitaste)
GitHub (azul si no lo visitaste)

:first-child y :last-child

li:first-child { background: #e3f2fd; }
li:last-child { background: #f3e5f5; }
  • Primer elemento (destacado)
  • Elemento normal
  • Elemento normal
  • Último elemento (destacado)

:nth-child(n) - Filas alternas

tr:nth-child(even) { background-color: #f5f5f5; }
Fila 1 (impar)
Fila 2 (par)
Fila 3 (impar)
Fila 4 (par)

:not() - Excluir elementos

.note:not(.important) { opacity: 0.7; }
Nota normal (más opaca)
Nota importante (más visible)
Otra nota normal (más opaca)

:disabled y :enabled

input:enabled { background-color: white; }
input:disabled { background-color: #f5f5f5; }

:checked - Elementos seleccionados

input[type="checkbox"]:checked + label { color: #667eea; }
